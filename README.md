# JailTeleport
ルール違反者を牢屋に投げ入れた上でLobiで通知  
違反プレイヤーを牢屋に投げ入れゲームモードをアドベンチャーモードに強制変更からのLobiの指定グループに自動投稿するだけのプラグインです。  
  
# ライセンス
